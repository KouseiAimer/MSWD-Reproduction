# MSWD 单独效用实验结果

## 1. 输出概况

本次实验已经完成 6 个 beta 设定下的 MSWD bootstrap 分析。

每个 beta 生成一组 Model A 数据，并进行 `B = 500` 次 bootstrap，因此结果文件中共有：

```text
6 * 500 = 3000 条 bootstrap 明细结果
```

输出文件如下：

```text

```

## 2. 汇总结果

显著性水平为：

```text
alpha = 0.05
```

拒绝规则为：

```text
observed_Tn > threshold
```

其中 `threshold` 是 500 次 bootstrap 统计量的 95% 分位数。

| beta | signal | B | observed_Tn | threshold | p-value | reject | true label | bootstrap mean | bootstrap sd |
|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|
| 0.0 | 0.00 | 500 | 11.0247 | 10.8511 | 0.024 | 1 | 0 | 9.3777 | 0.9748 |
| 0.2 | 0.16 | 500 | 5.8077 | 6.8852 | 0.410 | 0 | 1 | 5.5778 | 0.8444 |
| 0.4 | 0.32 | 500 | 6.4115 | 6.8455 | 0.162 | 0 | 1 | 5.6296 | 0.8021 |
| 0.6 | 0.48 | 500 | 4.4430 | 3.9048 | 0.004 | 1 | 1 | 3.2845 | 0.3485 |
| 0.8 | 0.64 | 500 | 6.9155 | 4.0032 | 0.000 | 1 | 1 | 3.2842 | 0.3895 |
| 1.0 | 0.80 | 500 | 9.7981 | 3.9948 | 0.000 | 1 | 1 | 3.2880 | 0.3871 |

这里的 `true label` 表示数据生成机制中是否真的存在两组分布差异：

```text
beta = 0      -> true label = 0
beta > 0      -> true label = 1
```

## 3. 结果解释

### beta = 0

当 `beta = 0` 时，理论上两组分布没有差异，原假设成立。

本次实验中：

```text
observed_Tn = 11.0247
threshold = 10.8511
p-value = 0.024
reject = 1
```

因此该次实验拒绝了原假设。这属于一次假阳性结果。

需要注意，这里只生成了一组 `beta = 0` 的数据，并不是 500 次独立重复模拟。由于显著性水平是 0.05，在原假设成立时，单次实验本身就可能以约 5% 的概率出现拒绝。因此这个结果不能直接说明 size 失控，只能说明本次抽到的数据和 bootstrap 结果导致了拒绝。

### beta = 0.2 和 beta = 0.4

当 `beta = 0.2` 和 `beta = 0.4` 时，两组分布已经存在差异，但差异较弱。

本次实验中：

```text
beta = 0.2: observed_Tn = 5.8077, threshold = 6.8852, reject = 0
beta = 0.4: observed_Tn = 6.4115, threshold = 6.8455, reject = 0
```

这说明在较弱信号下，当前这两组样本上的 MSWD 统计量还没有超过 bootstrap 阈值，因此没有检测到显著差异。

### beta = 0.6、0.8 和 1.0

当 beta 增大到 `0.6` 及以上时，MSWD 检验开始明显拒绝原假设。

结果为：

```text
beta = 0.6: observed_Tn = 4.4430, threshold = 3.9048, reject = 1
beta = 0.8: observed_Tn = 6.9155, threshold = 4.0032, reject = 1
beta = 1.0: observed_Tn = 9.7981, threshold = 3.9948, reject = 1
```

尤其在 `beta = 0.8` 和 `beta = 1.0` 时，观测统计量明显高于 bootstrap 分布的 95% 阈值，p-value 约为 0，说明 MSWD 对较强的均值衰减信号具有明显检测能力。

## 5. 总体结论

在本次单组样本加 bootstrap 的实验设置下，MSWD 方法对较强的 Model A 均值衰减信号有较明显的检测效果。

对于较弱信号 `beta = 0.2` 和 `beta = 0.4`，本次实验没有拒绝原假设，说明单组样本下的观测统计量仍然落在 bootstrap 零分布的非极端区域。

对于 `beta = 0.6`、`0.8` 和 `1.0`，观测统计量超过 bootstrap 阈值，说明 MSWD 能够检测到这些较强信号下的两样本分布差异。

如果要估计论文 Figure 1 中的 power 曲线，则还需要对每个 beta 进行多次独立 Monte Carlo 重复；当前实验主要用于观察单个样本下 MSWD 统计量与其 bootstrap 零分布之间的关系。
